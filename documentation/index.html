Application Style Editor

Application style editor is a program which provides functionality for changing various styles of differnt apps, like adjusting colors or layouts.

  • Editors:
  • 1. Color Editor
    • 1.1 Start Color Editor

      • When you start the program and choose the color editor there will load the default colors of the current app but only if the app has run at least once before that. You can keep the app and the editor running at the same time for preview purposes. If the app has not been running before that you have two other options.


    • 1.2 Load Document

      • The first option is to load a document using the load button and when the document is loaded you can edit it's colors. Every element in the app has it's own unique name, it is not possible to have elements with the same name, for example: if the name is "background" with color green, there will not be another element called background. This is important thing to mention especially for the second option - new document.


    • 1.3 New Document

      • The new document option is useful when you want to constuct a new document with unique names for every element and to apply colors for your application. It is important to know that you can enter only valid color names or hex codes, for example:red or #ffffff; RGB is not supported. You can add new lines and remove lines. The maximum number of lines which can be added is 200.


    • 1.4 Saving Changes

      • You can save changes at any time, you just need to click the save button. You can save changes in the load section or in new document section. You should know that although you can see the changes live on the app while editing the colors, these changes are not saved automatically!


    • 1.5 Preview

      • You can preview the changes live on the app if it is running on a side but it is not required. There are also painted rectangles which represent the colors in the editor, these rectangles help to determine the color codes and names.
